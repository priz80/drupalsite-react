Вот пошаговая инструкция, как запустить существующий проект:
Локальный запуск проекта с GitHub


1. Перейдите в папку куда будет клонироваться проект
введите в терминале: cd буква диска:\путь к папке
 
2. Клонируйте репозиторий:
введите в терминале: git clone https://github.com/priz80/drupalsite-react.git

3. Перейдите в папку проекта: cd drupalsite-react

4. Установите зависимости
введите в терминале: npm install (или yarn install), чтобы установить все необходимые библиотеки.

5. Запустите проект
введите в терминале: npm start. 

Приложение откроется в браузере по адресу http://localhost:3000
